Question 1 =>

var : var has function or global scope, can be hoisted, and is reassignable.
let : let has block scope, cannot be hoisted, and is reassignable.
const : const has block scope, cannot be hoisted, and holds a constant value after assignment.

Question 2 :

undefined is the default value for uninitialized variables or the absence of a value, while null is an explicit value indicating intentional absence of any object value.
